# ziyan
#### 概述

模仿知乎的一个小程序，用于任何使用微信的人都可以发文和互相讨论，还有许多功能未实现，欢迎大家讨论和交流。

#### 功能

1. 发表文章
2. 评论
3. 登录
4. 查看文章
5. 时间线更新
6. 匿名讨论和发文章
